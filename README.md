# pm_posneg
개인형 이동장치에 대한 여론 분석 프로젝트

**인 턴 근 무 기 관 :** 도로교통공단 서울지부
 
**기 간 :** 2021.09.06 ~ 2022.02.28
 
**팀 명 :** 위드 서울
 
**팀 원 :** 박호준, 이주천, 장효주

**수 상 :** 도로교통공단 이사장상(우수상)


**분 석 주 제 :** 개인형이동장치(PM)에 대한 다각적인 분석 

**1. PM에 대한 여론 분석**

2. PM, 이륜차, 자전거를 분류하는 로지스틱 회귀 모형 구축 및 관련 요인 탐색
3. PM 주행 위반 알고리즘 개발
